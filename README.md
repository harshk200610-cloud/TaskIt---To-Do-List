# TaskIt

A modern Todo Application built using **HTML**, **CSS**, and **Vanilla JavaScript**.

The project focuses on learning JavaScript fundamentals by building a real-world application instead of following isolated examples. It demonstrates DOM manipulation, event handling, state management, local storage, and responsive UI development.

---

## Preview

TaskIt allows users to

- Add tasks
- Assign categories
- Set priorities
- Select due dates
- Mark tasks as completed
- Delete tasks
- Filter tasks
- Automatically save tasks using Local Storage

---

## Features

### Task Management

- Create new tasks
- Delete tasks
- Mark tasks as complete
- Active task counter

### Categories

- Work
- Personal
- Academic
- Urgent

### Priority Levels

- Low
- Medium
- High

Each priority is represented using a different colored border.

### Due Date

Each task includes a due date.

### Filtering

- All Tasks
- Active Tasks
- Completed Tasks

### Persistent Storage

Tasks are automatically saved in Local Storage and remain available after refreshing the page.

### Responsive Design

The application works on desktop and mobile devices.
